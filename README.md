マークダウンベースのテキストエディタです。

# 文法
見出し：
`/#{1,6} (?<content>.*)(?<alias>\[.*\])/`

番号付きリスト：
`/\$ {1,6}(?<content>.*)/` 

番号なしリスト：
`/+ {1,6}(?<content>.*)/`

太字：
`/\*(?<content>.*)\*/` 

斜体：
`/\/(?<content>.*)\//`

上付き文字：
`/\^(?<content>.*)\^/`

下付き文字：
`/_(?<content>.*)_/`

ヘッダーエイリアス参照
`/\[header\.(?<aliasName>.*\]/`
